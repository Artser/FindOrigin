# Как настроить биллинг в Google Cloud Console

## Способ 1: Через меню навигации (Рекомендуется)

1. **Откройте Google Cloud Console:**
   - Перейдите на: https://console.cloud.google.com/
   - Убедитесь, что выбран правильный проект

2. **Найдите меню навигации:**
   - В левом верхнем углу нажмите на **три горизонтальные линии** (☰) - это меню
   - Или найдите иконку меню рядом с логотипом Google Cloud

3. **В меню найдите раздел "Billing":**
   - Прокрутите меню вниз
   - Найдите раздел **"Billing"** (Биллинг)
   - Нажмите на него

4. **Если биллинг не настроен:**
   - Вы увидите сообщение "No billing accounts"
   - Нажмите **"Link a billing account"** (Привязать счет биллинга)
   - Или **"Create billing account"** (Создать счет биллинга)

## Способ 2: Прямая ссылка

Откройте напрямую:
- **Биллинг:** https://console.cloud.google.com/billing
- **Создание биллинга:** https://console.cloud.google.com/billing/create

## Способ 3: Через поиск

1. В верхней части Google Cloud Console найдите **поисковую строку** (Search)
2. Введите: **"billing"** или **"биллинг"**
3. Выберите **"Billing"** из результатов поиска

## Пошаговая настройка биллинга

### Шаг 1: Откройте раздел биллинга

Используйте один из способов выше, чтобы открыть раздел Billing.

### Шаг 2: Создайте или привяжите счет биллинга

Если у вас еще нет счета биллинга:

1. Нажмите **"Create billing account"** (Создать счет биллинга)
   - Или **"Link a billing account"** (Привязать счет биллинга), если счет уже есть

2. Заполните форму:
   - **Account name** (Название счета): любое имя (например: "FindOrigin Billing")
   - **Country or region** (Страна или регион): выберите вашу страну
   - **Currency** (Валюта): выберите валюту (обычно определяется автоматически)

3. Нажмите **"Continue"** (Продолжить)

### Шаг 3: Добавьте способ оплаты

1. Выберите тип платежа:
   - **Credit or debit card** (Кредитная или дебетовая карта)
   - Или **Bank account** (Банковский счет) - если доступно в вашей стране

2. Заполните данные карты:
   - Номер карты
   - Срок действия
   - Имя держателя
   - CVV код
   - Адрес (если требуется)

3. Нажмите **"Submit"** (Отправить)

4. Подтвердите платеж (если требуется)

### Шаг 4: Привяжите счет к проекту

1. После создания счета биллинга вернитесь в раздел **Billing**

2. Найдите ваш проект в списке

3. Если проект не привязан:
   - Нажмите на проект
   - Выберите **"Change billing account"** (Изменить счет биллинга)
   - Выберите созданный счет биллинга
   - Нажмите **"Set account"** (Установить счет)

## Важная информация о биллинге

### Бесплатный тариф

⚠️ **Важно:** Настройка биллинга НЕ означает, что с вас будут списываться деньги!

- **Бесплатный лимит:** 100 запросов в день для Custom Search API
- **Деньги не списываются** на бесплатном тарифе
- **$200 кредитов** предоставляется на 30 дней для новых аккаунтов
- После использования кредитов бесплатные сервисы остаются бесплатными

### Когда списываются деньги?

Деньги списываются только если:
- Вы превысили бесплатный лимит
- Используете платные сервисы
- Явно выбрали платный тариф

### Настройка лимитов расходов

Чтобы избежать неожиданных расходов:

1. В разделе Billing найдите **"Budgets & alerts"** (Бюджеты и уведомления)
2. Создайте бюджет с лимитом (например, $0 или $5)
3. Настройте уведомления при достижении лимита

## Проверка статуса биллинга

После настройки биллинга:

1. Откройте: https://console.cloud.google.com/billing
2. Убедитесь, что ваш проект привязан к счету биллинга
3. Статус должен быть **"Active"** (Активен)

## Устранение проблем

### Не вижу раздел Billing в меню

**Решение:**
- Убедитесь, что вы вошли в правильный аккаунт
- Проверьте, что у вас есть права администратора проекта
- Попробуйте использовать прямую ссылку: https://console.cloud.google.com/billing

### Ошибка при добавлении карты

**Возможные причины:**
- Карта не поддерживается (проверьте список поддерживаемых карт)
- Неверные данные карты
- Проблемы с банком

**Решение:**
- Попробуйте другую карту
- Свяжитесь с поддержкой Google Cloud

### Биллинг настроен, но API все еще не работает

**Проверьте:**
1. Custom Search API включен в проекте
2. API ключ привязан к Custom Search API
3. Не превышен дневной лимит (100 запросов)
4. Подождите 5-10 минут после настройки биллинга (изменения применяются не мгновенно)
