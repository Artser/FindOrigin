# Инструкция по созданию биллинг-аккаунта в Yandex Cloud

## Важно

⚠️ **Без активного биллинг-аккаунта API будет возвращать ошибку 403 "Permission denied"!**

Даже для бесплатного использования биллинг-аккаунт **обязателен**. Это не означает, что с вас будут списываться деньги - просто требуется привязка способа оплаты.

## Шаг 1: Вход в Yandex Cloud

1. Перейдите на [Yandex Cloud Console](https://console.cloud.yandex.ru/)
2. Войдите используя ваш Яндекс аккаунт
3. Если у вас нет аккаунта, создайте его на [yandex.ru](https://yandex.ru/)

## Шаг 2: Переход в раздел "Биллинг"

### Способ 1: Через меню навигации

1. В левой части экрана найдите меню навигации
2. Найдите раздел **"Биллинг"** (Billing)
3. Нажмите на него
4. В подменю выберите **"Биллинг-аккаунты"** (Billing accounts)

### Способ 2: Прямая ссылка

Откройте напрямую:
- **Биллинг-аккаунты:** https://console.cloud.yandex.ru/billing

### Способ 3: Через поиск

1. В верхней части консоли найдите поисковую строку
2. Введите: **"биллинг"** или **"billing"**
3. Выберите **"Биллинг-аккаунты"** из результатов

## Шаг 3: Создание нового биллинг-аккаунта

1. В разделе "Биллинг-аккаунты" нажмите кнопку **"Создать биллинг-аккаунт"** или **"Create billing account"**

2. Заполните форму создания биллинг-аккаунта:

   **Название биллинг-аккаунта:**
   - Введите любое название (например: "FindOrigin Billing")
   - Это внутреннее название для вашего удобства

   **Тип плательщика:**
   - **Физическое лицо** - если вы создаете для личного использования
   - **Юридическое лицо** - если для компании
   - Выберите подходящий вариант

   **Страна:**
   - Выберите вашу страну из списка
   - Для России выберите "Россия"

3. Нажмите **"Создать"** или **"Create"**

## Шаг 4: Привязка способа оплаты

После создания биллинг-аккаунта нужно привязать способ оплаты:

1. Выберите созданный биллинг-аккаунт из списка

2. Нажмите **"Привязать способ оплаты"** или **"Link payment method"**

3. Выберите тип платежа:
   - **Банковская карта** (Visa, MasterCard, МИР)
   - **Электронный кошелек** (Яндекс.Деньги, WebMoney и т.д.)
   - **Банковский перевод** (для юридических лиц)

4. Заполните данные:
   - Для карты: номер, срок действия, CVV, имя держателя
   - Для электронного кошелька: номер кошелька
   - Для банковского перевода: реквизиты счета

5. Подтвердите привязку

⚠️ **Важно:** Привязка способа оплаты не означает списание денег. Деньги списываются только при использовании платных ресурсов.

## Шаг 5: Привязка биллинг-аккаунта к облаку

1. Вернитесь в главное меню Yandex Cloud Console

2. Перейдите в раздел **"Облака"** (Clouds) или **"Все ресурсы"** (All resources)

3. Выберите ваше облако (или создайте новое, если его нет)

4. В настройках облака найдите раздел **"Биллинг"** или **"Billing"**

5. Нажмите **"Привязать биллинг-аккаунт"** или **"Link billing account"**

6. Выберите созданный биллинг-аккаунт из списка

7. Нажмите **"Привязать"** или **"Link"**

## Шаг 6: Активация биллинг-аккаунта

1. Вернитесь в раздел **"Биллинг"** → **"Биллинг-аккаунты"**

2. Найдите ваш биллинг-аккаунт в списке

3. Проверьте статус:
   - **"Активен"** (Active) - биллинг-аккаунт готов к использованию
   - **"Неактивен"** (Inactive) - требуется активация

4. Если статус "Неактивен":
   - Нажмите на биллинг-аккаунт
   - Нажмите **"Активировать"** или **"Activate"**
   - Подтвердите активацию

5. После активации статус должен измениться на **"Активен"**

## Проверка настройки

После всех шагов проверьте:

1. ✅ Биллинг-аккаунт создан
2. ✅ Способ оплаты привязан
3. ✅ Биллинг-аккаунт привязан к облаку
4. ✅ Статус биллинг-аккаунта: **"Активен"**

## Бесплатный тариф Yandex Cloud

### Что входит в бесплатный тариф:

- **$300 стартовых кредитов** на 60 дней
- **Бесплатные квоты** на некоторые сервисы
- **Бесплатный доступ** к базовым функциям

### Когда списываются деньги:

Деньги списываются только если:
- Вы превысили бесплатные квоты
- Используете платные сервисы
- Истекли стартовые кредиты и вы продолжаете использовать платные ресурсы

### Настройка лимитов расходов:

Чтобы избежать неожиданных расходов:

1. В разделе "Биллинг" найдите **"Бюджеты"** (Budgets)
2. Создайте бюджет с лимитом (например, 0₽ или 100₽)
3. Настройте уведомления при достижении лимита

## Устранение проблем

### Ошибка: "Permission denied" (403)

**Причины:**
- Биллинг-аккаунт не создан
- Биллинг-аккаунт не привязан к облаку
- Биллинг-аккаунт не активирован
- Способ оплаты не привязан

**Решение:**
1. Проверьте все шаги выше
2. Убедитесь, что статус биллинг-аккаунта "Активен"
3. Проверьте, что биллинг-аккаунт привязан к вашему облаку

### Не вижу раздел "Биллинг"

**Решение:**
- Убедитесь, что вы вошли в правильный аккаунт
- Проверьте права доступа (нужны права администратора)
- Попробуйте прямую ссылку: https://console.cloud.yandex.ru/billing

### Ошибка при привязке способа оплаты

**Возможные причины:**
- Неверные данные карты
- Карта не поддерживается
- Проблемы с банком

**Решение:**
- Проверьте данные карты
- Попробуйте другую карту
- Свяжитесь с поддержкой Yandex Cloud

### Биллинг настроен, но API все еще не работает

**Проверьте:**
1. Подождите 5-10 минут после настройки (изменения применяются не мгновенно)
2. Проверьте, что API включен в проекте
3. Проверьте права доступа к API
4. Проверьте квоты и лимиты
