# Бесплатные email сервисы для Azure

## Важно: Для Azure нужен Microsoft аккаунт

Azure Portal работает только с **Microsoft аккаунтами**. Это означает, что нужен email от Microsoft или аккаунт, привязанный к Microsoft.

## Лучший вариант: Outlook.com (Рекомендуется) ⭐

**Outlook.com** - это бесплатный email сервис от Microsoft, который идеально подходит для Azure.

### Преимущества:
- ✅ Бесплатный
- ✅ Создается за 2-3 минуты
- ✅ Гарантированно работает с Azure Portal
- ✅ Полный доступ ко всем Microsoft сервисам
- ✅ Американский сервис (Microsoft - американская компания)

### Как создать:

1. Перейдите на: https://outlook.com/
2. Нажмите **"Create free account"** или **"Создать бесплатную учетную запись"**
3. Выберите имя пользователя (например: `findorigin2024`)
4. Выберите домен: **`@outlook.com`** (или `@hotmail.com`, `@live.com`)
5. Создайте пароль
6. Заполните данные и подтвердите регистрацию

**Готово!** Теперь у вас есть Microsoft аккаунт, который работает с Azure.

## Альтернативные варианты

### Вариант 1: Использовать существующий email для Microsoft аккаунта

Можно создать Microsoft аккаунт, используя ваш существующий email (mail.ru, yandex.ru, gmail.com и т.д.):

1. Перейдите на: https://account.microsoft.com/
2. Нажмите **"Sign up"** или **"Создать учетную запись"**
3. Выберите **"Use a phone number or get a new email address"**
4. Выберите **"Use your email instead"**
5. Введите ваш существующий email
6. Создайте пароль для Microsoft аккаунта
7. Подтвердите email

⚠️ **Важно:** Этот способ может не работать так же надежно, как новый Outlook.com аккаунт.

### Вариант 2: Gmail (НЕ работает напрямую)

**Gmail** - это Google сервис, он НЕ является Microsoft аккаунтом и **не работает с Azure Portal напрямую**.

Однако, можно:
- Использовать Gmail для создания Microsoft аккаунта (как описано выше)
- Или использовать Gmail для Google Cloud Console (альтернатива Azure)

### Вариант 3: Yahoo Mail (НЕ работает напрямую)

**Yahoo Mail** также не является Microsoft аккаунтом и не работает с Azure.

## Рекомендация

**Создайте новый аккаунт на Outlook.com** - это:
- Самый надежный вариант
- Занимает 2-3 минуты
- Гарантированно работает с Azure
- Бесплатный навсегда

## Быстрая инструкция

1. Откройте: https://outlook.com/
2. Нажмите **"Create free account"**
3. Введите имя: `findorigin2024` (или любое другое доступное)
4. Выберите `@outlook.com`
5. Создайте пароль
6. Заполните данные
7. Подтвердите регистрацию
8. Готово! Используйте этот email для Azure Portal

## После создания аккаунта

1. Войдите в [Azure Portal](https://portal.azure.com/) с новым Outlook.com аккаунтом
2. Создайте бесплатный Azure аккаунт
3. Создайте ресурс Bing Search v7
4. Получите API ключ
5. Добавьте в `.env.local`:
   ```env
   BING_SEARCH_API_KEY=ваш_ключ
   ```
